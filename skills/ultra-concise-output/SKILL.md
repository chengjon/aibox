# Ultra-Concise Output Style

超简洁样式专为效率而设计，提供最少的信息量但保持最大的实用性。这是处理日常任务和快速查询的理想选择。

## 核心特性

- ✅ **最小文本**：去除所有不必要的词汇
- ✅ **最大效率**：直接给出答案和解决方案
- ✅ **命令导向**：优先提供可执行的命令
- ✅ **代码示例**：直接可用的代码片段
- ✅ **要点总结**：只保留最关键的信息
- ✅ **行动导向**：明确的下一步操作

## 使用方法

激活超简洁输出样式：

```
/output-style ultra-concise
```

然后提出您的问题或需求。

## 输出示例

### 问题：如何创建React组件？

```
/output-style ultra-concise

// 函数组件
function Button({ onClick, children }) {
  return <button onClick={onClick}>{children}</button>
}

// 类组件
class Button extends React.Component {
  render() {
    return <button onClick={this.props.onClick}>
      {this.props.children}
    </button>
  }
}
```

## 变体选择

根据具体需求选择合适的输出样式：

- **ultra-concise**: 最少信息，最大效率
- **bullet-points**: 简洁但结构化
- **standard**: 平衡信息量和可读性
- **detailed**: 完整解释和背景
- **genui**: 视觉化展示

## 相关技能

- `bullet-points` - 结构化但简洁
- `code-snippets` - 代码导向的简洁格式
- `quick-reference` - 快速查询格式
- `command-cheatsheet` - 命令参考格式
