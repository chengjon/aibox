# GenUI Output Style

GenUI是Claude-Kits中最强大的输出样式，能够生成包含完整内嵌样式的HTML输出。这使得响应内容具有专业的外观，可以在浏览器中立即预览。

## 核心特性

- ✅ **完整HTML结构**：生成语义化的HTML5标签
- ✅ **内嵌CSS样式**：所有样式直接嵌入，无需外部依赖
- ✅ **即时预览**：在浏览器中直接查看效果
- ✅ **响应式设计**：适配不同屏幕尺寸
- ✅ **代码高亮**：支持多种编程语言的语法高亮
- ✅ **表格样式**：美观的表格和数据展示
- ✅ **颜色主题**：现代化的配色方案

## 使用方法

激活GenUI输出样式：

```
/output-style genui
```

然后正常描述您希望输出的内容，Claude会使用GenUI样式格式化响应。

## 输出示例

```html
<div class="genui-container">
  <h1 class="genui-title">项目状态报告</h1>
  <div class="genui-card">
    <h2>开发进度</h2>
    <ul class="genui-list">
      <li>✅ 前端组件完成</li>
      <li>🔄 API集成进行中</li>
      <li>⏳ 测试用例编写</li>
    </ul>
  </div>
</div>
```

## 自定义配置

GenUI样式支持以下自定义选项：

- **主题色彩**：蓝色、绿色、紫色、橙色
- **布局风格**：紧凑、标准、宽松
- **组件大小**：小、中、大
- **字体样式**：默认、编程、正式

## 注意事项

- 生成的内容包含完整的HTML结构，建议在安全的环境中查看
- 复杂的内容可能需要手动调整样式以达到最佳效果
- 在某些终端环境中可能无法直接预览HTML效果
- 大型表格可能在移动设备上需要横向滚动
